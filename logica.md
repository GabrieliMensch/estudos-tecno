## Estruturas condicionais (if/else)

```python
# Verificar se um número é par ou ímpar
numero = 10

if numero % 2 == 0:
    print("O número é par")
else:
    print("O número é ímpar")
# Exibir números de 1 a 5
for i in range(1, 6):
    print(i)
# Função que soma dois números
def soma(a, b):
    return a + b

resultado = soma(5, 7)
print("Resultado:", resultado)
